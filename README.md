# Template Full Stack Application

## Prerequisites
* **Node.js** with **Yarn**
* **Python 3 Fast API**
* **uv**. Follow the installation instructions to install the latest stable version [here](https://docs.astral.sh/uv/getting-started/installation/)

## Development Practice
* It's strongly encouraged to follow the folder/file conventions for both the FE and BE
* [FastAPI Docs](https://fastapi.tiangolo.com/tutorial/bigger-applications/#another-module-with-apirouter)
* [React Docs](https://react.dev/)
* [Vite Docs](https://vite.dev/guide/)

## Quick Start

### 1. Install Dependencies
Run this single command to setup both Frontend (Node) and Backend (Python/uv) environments:

```bash
yarn install:all
```

### 2. Run the app

#### Method 1: Two Terminal Window view
In one terminal window, run `yarn frontend`

In another terminal window, run `yarn backend`

#### Method 2: One-go
If you want to spin up both frontend and backend but without two separate commands, run `yarn dev`
